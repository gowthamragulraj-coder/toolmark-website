Tool Mark Identification System

This website is a prototype forensic tool developed to assist in the identification and understanding of tool marks found at crime scenes.

The system allows users to upload an image of a suspected tool mark and view common forensic features used in tool mark examination.

Objectives

  • To assist in tool mark identification
  • To support forensic education
  • To provide a low-cost digital forensic tool

Upload Tool Mark Image

Tool Mark Features

  • Class characteristics
  • Individual characteristics
  • Striation patterns
  • Depth variation
  • Direction of force

Forensic Significance

Tool mark examination plays a vital role in crime scene reconstruction and weapon identification in forensic investigations.
